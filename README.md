# mini-vue3

一步步实现一个mini-vue3。mini-Vue3是一个参照Vue3实现的mini框架。该框架目前实现了响应式、运行时等核心功能，后续也会融入一些有趣的其他功能,不限于来自vue3。

## 功能

- 响应式
- 运行时
- 模板编译（即将实现）

## 安装依赖

使用以下命令安装依赖：

```
yarn install
```

## 打包

打包全部模块：

```
yarn build
```

## 核心功能实现进度如下
- reactivity
- runtime-core
- runtime-dom
- compiler-core
- compiler-dom
